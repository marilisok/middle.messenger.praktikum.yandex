## Установка и запуск

### Установка

Установка зависимостей проекта:

```bash
npm i
```

### Сборка и запуск

```bash
npm run build
```

Выполняется сборка и запуск статического сервера на Express, на порту 3000. http://localhost:3000:

```bash
npm run start
```
